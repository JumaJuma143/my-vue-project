1) скачати vs code чи будь-який інший редактор коду
2) скачати та налаштувати node. перевірити в терміналі редактора через node -v та npm -v
3) при потребі встановити даний проект npm install leaflet
4) запустити проект через команду в терміналі npm run dev
5) перейти по посиланню, що сформується в терміналі
